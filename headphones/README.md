
# Project Title
HEADPHONES
## Description

This project is a responsive web page designed to provide an optimal user experience across various devices. The layout adapts seamlessly to different screen sizes, ensuring that users can interact with the content easily, whether on a desktop or mobile device.

## Features

- **Responsive Design**: The web page automatically switches to a mobile version when the screen width is 480px or less.
- **Interactive Elements**:
  - **Links**: Change color to `#FF6565` on hover and active states.
  - **Buttons**: Reduce opacity to `0.9` on hover and active states.
- **Content Layout**: The maximum width of the content is set to `1000px`, ensuring it is centered on the page for a clean and organized appearance.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ErnstBanks/alx_html_css.git
   ```
2. Navigate to the project directory:

   cd projectname
   ```

## Usage

- Simply load the web page in a browser to see the responsive design in action.
- Interact with links and buttons to observe the hover effects.
